Vipiend App

Vipiend is a VPN application that implements OpenVPN as a service. This app connects you to the fastest server for the best internet experience.

Hafalin App

Hafalin applies Speech To Text technology with the One Day One Ayat Method to facilitate independent Quran memorization submissions. Additionally, this application also provides the entire Quran text that can be read and saved for bookmarking purposes. This app was build with Kotlin and integration with Google Speech API for speech recognition.

PlastiCode App

PlastiCode is an application that educates about the types of plastic along with their lifespan and level of danger. There are three main features in this application: plastic type detection, recommendations for plastic alternatives, and recommendations for plastic recycling locations. PlastiCode is built with Kotlin, TensorFlow Lite, and implements the Clean Code Architecture. This application was developed as the Capstone Project Bangkit 2023.

UIN Jakarta IT Services Website

This website developed to provide various services related to the Informatics Engineering program at UIN Jakarta. It aims to streamline and enhance the administrative processes within the program. This website build with Laravel framework and MySQL.
